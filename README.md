  <h2 style="color:#1f77b4;">ğŸ“š Dataset</h2>
  <p>
    The <b>FIDAC dataset</b> is available on <a href="https://ieee-dataport.org/documents/fidac-forged-images-detection-and-classification">IEEE Dataport</a> and contains original camera-clicked images along with their tampered versions.
